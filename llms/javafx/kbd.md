# PrizmKbd (JavaFX)

A keyboard key indicator — monospace text on a muted key-cap chip (Label subclass).

In spec parity with the React `Kbd` (`components/ui/kbd.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`kbd` key and on the components page at `/components/kbd`.

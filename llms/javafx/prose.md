# PrizmProse (JavaFX)

A reading-width container that stacks PrizmHeading / PrizmText with paragraph spacing. JavaFX has no HTML to style, so it is a layout container (VBox), not a rich-text styler.

In spec parity with the React `Prose` (`components/ui/prose.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`prose` key and on the components page at `/components/prose`.

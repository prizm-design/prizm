# PrizmTabs (JavaFX)

Tabbed sections — thin TabPane subclass; muted labels with accent + muted-fill on the selected tab. Add Tabs via getTabs().

In spec parity with the React `Tabs` (`components/ui/tabs.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`tabs` key and on the components page at `/components/tabs`.

# PrizmPagination (JavaFX)

Previous / numbered pages / Next — composite HBox of PrizmButtons (active outlined, others ghost). setOnPageChange for the callback. Shows all pages (no ellipsis).

In spec parity with the React `Pagination` (`components/ui/pagination.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`pagination` key and on the components page at `/components/pagination`.

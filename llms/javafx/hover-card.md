# PrizmHoverCard (JavaFX)

A hover-triggered floating preview — built on PopupControl (inherits the scene theme), reuses the .prizm-popover surface. Use attach(owner) to show on mouse-enter and hide on exit.

In spec parity with the React `Hover Card` (`components/ui/hover-card.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`hover-card` key and on the components page at `/components/hover-card`.

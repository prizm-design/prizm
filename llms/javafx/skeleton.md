# PrizmSkeleton (JavaFX)

A loading placeholder — a muted Region that pulses (indefinite opacity animation, mirroring animate-pulse). Size via the constructor or setPrefSize.

In spec parity with the React `Skeleton` (`components/ui/skeleton.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`skeleton` key and on the components page at `/components/skeleton`.

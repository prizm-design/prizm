# PrizmBreadcrumb (JavaFX)

A breadcrumb trail — composite HBox of muted clickable crumbs (addCrumb) separated by chevrons, ending in a non-link current page (addCurrent).

In spec parity with the React `Breadcrumb` (`components/ui/breadcrumb.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`breadcrumb` key and on the components page at `/components/breadcrumb`.

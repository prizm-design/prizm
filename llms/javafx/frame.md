# PrizmFrame (JavaFX)

A max-width, padded content container (VBox) with Padding and MaxWidth presets. The web centres with mx-auto; in JavaFX place it in a centring parent.

In spec parity with the React `Frame` (`components/ui/frame.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`frame` key and on the components page at `/components/frame`.

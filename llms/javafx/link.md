# PrizmLink (JavaFX)

A hyperlink — Hyperlink subclass restyled to accent text, underline-on-hover, no focus border. Wire navigation with setOnAction.

In spec parity with the React `Link` (`components/ui/link.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`link` key and on the components page at `/components/link`.

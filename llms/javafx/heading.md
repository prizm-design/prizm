# PrizmHeading (JavaFX)

A heading — thin Label subclass with a semibold, tight type scale (Size XL4 36 / XL3 30 / XL2 24 / XL 20 / LG 18 / MD 16; default XL2). JavaFX has no semantic heading tags, so only the visual size applies.

In spec parity with the React `Heading` (`components/ui/heading.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`heading` key and on the components page at `/components/heading`.

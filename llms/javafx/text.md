# PrizmText (JavaFX)

Body text — thin Label subclass with chainable size / tone / weight presets (Size XS–LG, Tone DEFAULT/MUTED/SUBTLE, Weight NORMAL/MEDIUM/SEMIBOLD).

In spec parity with the React `Text` (`components/ui/text.tsx`). The full
Java API (constructors, enums, methods) is in `lib/javafx-api.ts` under the
`text` key and on the components page at `/components/text`.
